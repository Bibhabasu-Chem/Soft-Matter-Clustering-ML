# Soft-Matter-Clustering-ML 🤖

Exploratory Machine Learning analysis of particle configurations in model systems.

### 🎯 Project Aim
- Bridge classical statistical mechanics intuition with data-driven methods.
- Focus on **polymorph discrimination** and **interface characterization** in soft colloidal systems.

### 🛠️ Methodology
- **Techniques:** K-means and DBSCAN clustering.
- **Library:** Scikit-learn (Python).
- **Goal:** Identify structural patterns associated with phase behaviour.
